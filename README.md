# Task-4 Optimization Model

This project demonstrates how to solve a business optimization problem using Linear Programming and Python PuLP library.

## Objective
Maximize business profit using limited labor and machine resources.

## Tools Used
- Python
- PuLP
- Pandas
- Jupyter Notebook

## Folder Structure
- data: Input CSV file
- notebooks: Jupyter notebook
- src: Python script
- results: Output file
- report: Final report

## How to Run

1. Install dependencies:
pip install -r requirements.txt
